Asking liaison lead to review support requests

Send this email to a liaison lead to share support requests from local organisations ready for their review.

Checklist

1. Copy and paste example text below into the email.

2. Copy and paste the request reports into the email.

3. Add the multiagency cell leads contact list.



Dear Liaison Lead

Re: Requests ready for local review

Please review the following requests for support made by local organisations in your local area.

Request ID: [ ]

Request link: [ ]

1. Check there is no information missing

If information is missing, for example no details on what type of support required, contact the local organisation directly to find out more.

When you have received the missing information, add it into the relevant columns.

2. Consider if the request can be met locally

Yes: select 'Exploring local' in the SharePoint status column.

No: select 'Escalate to MAC' in the SharePoint status column.

If the request becomes urgent, notify the multiagency cell lead.

For support editing records in SharePoint, contact an information manager.

Many thanks,

Information Manager

'Request support as a voluntary or community organisation' service



We recommend you set up and save this example response as a draft email, and copy it from there each time you need to use it.
