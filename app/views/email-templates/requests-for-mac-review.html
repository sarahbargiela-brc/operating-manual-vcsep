Asking multiagency cell members to review support requests

Send this email to a multiagency cell distribution list to share support requests from local organisations ready for their review.

Checklist

1. Copy and paste example text below into the email.

2. Copy and paste the request reports into the email.

3. Add the multiagency cell distribution list.



Dear [Name of Cell] members

Re: Requests ready for multiagency cell review

Please review the following requests for support made by local organisations in the [name of cell] area.

Request ID: [ ]

Request link: [ ]

If the request is urgent, it can be brokered ahead of the next multiagency cell meeting.

Follow the guidance on handling urgent requests.

For support editing records in SharePoint, contact an information manager.

Many thanks,

Information Manager

'Request support as a voluntary or community organisation' service



We recommend you set up and save this example response as a draft email, and copy it from there each time you need to use it.
