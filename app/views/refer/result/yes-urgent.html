What you should do

Refer the person to local BRC area teams for support.

1. Find out from the caller:

  1. What medication they need, if any
  2. How many in the household need food, if any
  3. Any dietary requirements
  4. Any specific needs. For example nappies

2. Assess if any further support needed, for example psychosocial care.

3. Make the referral using the form to record information from the call
